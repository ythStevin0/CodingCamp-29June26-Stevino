# To-Do List Life Dashboard

Ini adalah proyek akhir (mini project) untuk kursus rekayasa perangkat lunak CodingCamp. Proyek ini merupakan dasbor web sederhana yang membantu pengguna mengatur hari mereka, mencakup jam digital, daftar tugas, pengatur waktu fokus (Pomodoro), dan tautan cepat.

## 🔗 Live Demo
*(Jangan lupa tambahkan link GitHub Pages kamu di sini, contoh: `https://ythStevin0.github.io/CodingCamp-29June26-Stevino/`)*

## 🚀 Fitur Utama (MVP)
- **Sapaan & Waktu**: Menampilkan jam, tanggal, dan sapaan yang dinamis berdasarkan waktu saat ini.
- **Timer Fokus**: Timer Pomodoro (default 25 menit) dengan fitur mulai, jeda, dan ulangi. Dilengkapi dengan notifikasi suara dan pop-up saat waktu habis.
- **Daftar Tugas (To-Do List)**: Tambah, edit, tandai selesai, dan hapus tugas. Dilengkapi dengan filter (Semua, Aktif, Selesai).
- **Tautan Cepat (Quick Links)**: Menyimpan dan membuka website favorit dengan cepat.
- **Penyimpanan Lokal (Local Storage)**: Semua data (tugas, tautan, preferensi) disimpan secara lokal dan aman di dalam browser.

## 🏆 Tantangan Tambahan yang Diselesaikan
Proyek ini berhasil memenuhi **seluruh 5 tantangan tambahan** dari batas minimum 3:
1. ✅ **Light / Dark mode**: Bisa mengganti tema dasbor antara terang atau gelap.
2. ✅ **Custom name in greeting**: Nama pada sapaan dapat diklik, diubah, dan disimpan.
3. ✅ **Change Pomodoro time**: Bisa memilih durasi fokus antara 15, 25, atau 45 menit.
4. ✅ **Prevent duplicate tasks**: Mencegah penambahan tugas dengan nama yang sama (dilengkapi animasi *shake* untuk peringatan).
5. ✅ **Sort tasks**: Mengurutkan daftar tugas sesuai abjad A–Z atau Z–A.

## 🛠 Teknologi yang Digunakan
- HTML5 (Struktur halaman)
- CSS3 (Vanilla CSS untuk gaya dan *layout*)
- Vanilla JavaScript (Tanpa framework, menangani interaktivitas dan *Local Storage*)

## 📁 Struktur File
Sesuai dengan ketentuan tugas, proyek ini mengikuti struktur:
- 1 file CSS utama (`css/style.css`)
- 1 file JS utama (`js/script.js`)
- `index.html` dan `.kiro/` di dalam root direktori.
